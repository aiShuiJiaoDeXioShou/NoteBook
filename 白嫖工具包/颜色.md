# 好看的颜色

## 样例：

![image-20210925212756025](https://gitee.com/theCompassWillAlsoGetLost/typora-picture-resources2/raw/master/img/image-20210925212756025.png)

这张图的背景色：

~~~css
    color: #586e75;
    background-color: #fdf6e3;
	/* 这样一个效果让人看上去颜色变深 */
	li:hover{
		background: rgba(0, 0, 0, 5%);
	}
~~~
